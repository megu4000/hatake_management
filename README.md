# hatake_management
畑の管理を行うためのデモアプリ

## 概要
作物の状態や畑サイズなどの可変性を考慮した個人制作アプリ

## 使用技術
 - 言語：gdscript
 - フレームワーク：godot
 - 開発環境：Windows

## 動作環境
 - Windows

## アプリの実行方法
 - ダウンロードした「畑管理アプリ.zip」ファイルを解凍しhatake_demo.exeをダブルクリック．警告が出た場合、詳細→実行の順でクリック．

## 操作方法
 - 左クリックで畑情報確認．右クリックドラッグで畑エリア移動．矢印キーとマウスホイールでカメラ移動．
